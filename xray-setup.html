Установка VPN на новый сервер

Бинарник качается прямо на сервер с GitHub. Вводи ключи в правую панель — команды генерируются автоматически.

1
Скачать архив и распаковать
новый сервер
▾

Одна команда: скачивает архив с GitHub прямо на сервер и распаковывает. Windows вообще не нужен.

bash
wget https://github.com/dictovod/VPN/raw/refs/heads/main/xray-backup.tar.gz \
  -O /tmp/xray-backup.tar.gz && \
tar -xzvf /tmp/xray-backup.tar.gz -C / && \
mkdir -p /usr/local/etc/xray && \
echo "Готово — бинарник и сервис установлены"
i Если wget не установлен: apt-get install -y wget
2
Сгенерировать ключи
новый сервер
▾

Генерируем новые UUID, ключи Reality и Short ID. Вставь вывод в правую панель — команды обновятся автоматически.

bash
NEW_UUID=$(xray uuid) && \
NEW_KEYS=$(xray x25519) && \
PRIVATE_KEY=$(echo "$NEW_KEYS" | grep "PrivateKey" | awk '{print $2}') && \
PUBLIC_KEY=$(echo "$NEW_KEYS" | grep "Password" | awk '{print $2}') && \
SHORT_ID=$(openssl rand -hex 4) && \
echo "UUID: $NEW_UUID" && \
echo "Private key: $PRIVATE_KEY" && \
echo "Public key: $PUBLIC_KEY" && \
echo "Short ID: $SHORT_ID"

4
Запустить сервис
новый сервер
▾
bash
systemctl daemon-reload && \
systemctl enable xray && \
systemctl start xray && \
systemctl status xray
✓Должно быть: Active: active (running) и Xray X.X.X started
iЛоги в реальном времени: journalctl -u xray -f
